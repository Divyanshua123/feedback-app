# Feedback Collection System  A simple Node.js + Express app to collect and display user feedback.

A simple web-based Feedback Collection System built using HTML, CSS, and JavaScript.
Users can submit their feedback, and it will be displayed instantly on the same page.

## Features

* Collect user feedback (name + message)
* Display feedback instantly
* Clean and simple user interface
* No backend required
* Beginner-friendly project

## Project Structure

```
feedback-system/
│── index.html
```

## Technologies Used

* HTML
* CSS
* JavaScript (for dynamic updates)
